*****
	 LinkedIn Jobs Scraping Without Login

Set Up
#######

place `Chrome WebDriver <https://chromedriver.chromium.org/downloads>`_ in the same folder

.. code-block:: python

  from selenium import webdriver
  
  wd = webdriver.Chrome('chromedriver.exe')
